Customize Tables

Overview of gt

With gt you can construct a wide variety of useful tables with a cohesive set of table parts. Here is the main workflow to transform a data frame into a publication-ready table:

An gt object as various elements. These include the table header, the stub, the column labels and spanner column labels, the table body, and the table footer.

Our example

We are going to reproduce some of the tables of this paper Use of wireless motility capsule to determine gastric emptying and small intestinal transit times in critically ill trauma patients and create new ones using the study data in the smartpill dataset.

Load the libraries

# install.packages("gt")
library(gt)
library(tidyverse)
library(medicaldata)

A basic table

Now let’s create a summary table of demographics of study participants.

table1 <- smartpill  %>% 
  select(Group, Gender, Height, Weight, Age) %>%
  group_by(Group) %>%
    summarise(average_age = mean(Age),
              average_height = mean(Height),
              average_weight = mean(Weight))

table1
# A tibble: 2 × 4
  Group average_age average_height average_weight
  <fct>       <dbl>          <dbl>          <dbl>
1 0            40.2           176.           80.5
2 1            37.2           172.           77.2

We can turn this into a gt table with function gt():

table1_pub <-
  table1 %>%
  #make gt object
  gt() 

table1_pub

1 37.22989 171.6018 77.18989
